## To Add Your Profile Image:

1. **Save your profile image** with the name `profile.jpg` in the `public` folder
   - Path: `public/profile.jpg`
   - The image you provided will work perfectly!

2. **The full path should be:**

3. **Image Specifications:**

4. **Once added, the profile image will:**
   - Display on the left side of the hero section
   - Have a parallax effect (moves slower than scroll)
   - Have a golden border with professional shadow
   - Hover effect with elevation animation
